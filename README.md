# CharCounter - Windows Forms

Aplicación de escritorio que permite contar caracteres en un texto. Proporciona una interfaz simple e intuitiva para contar caracteres, con funcionalidades adicionales como copiar el texto y limpiar el campo de entrada.

## Requisitos
- .NET Framework
- Windows OS
- Visual Studio 2015 o superior (opcional, para compilar)

## Cómo ejecutar
1. Descargar o clonar el repositorio
2. Abrir `xprox_charCounter.sln` en Visual Studio
3. Compilar el proyecto
4. Ejecutar la aplicación (F5 o Debug > Start Debugging)
5. Alternativamente, ejecutar directamente el archivo `.exe` generado en el directorio `bin/Debug/`

## Uso
- Ingresa texto en el campo de entrada
- El contador mostrará automáticamente el número de caracteres
- Usa el botón "Copiar" para copiar el texto al portapapeles
- Usa el botón "Limpiar" para borrar el texto
